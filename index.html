    Introduction

    Ideology

    We do PHP. We like to do it using continuously evolving industry best practices, despite the language's shortcomings. We like to hone our work to a glamorous shine and share it with others to use and reuse. This is the fruit of our labour.

    Xi aims to be the de-facto destination to bootstrap your PHP development. In other words, if there's something architectural you need to construct every time you fire up a PHP project, this might be the right place for a generic implementation.

    Come and join us - contributions are welcome!

    Features

    Here's something the project can offer:

    • BSD v3 license: use the code freely, no strings attached.
    • Loose coupling: no huge download, cherry pick only the components you use.
    • Based on Github: easy to use, fork and contribute to.
    • No CLA requirement for contributors: that's not how we roll.
    • Corporate backing: sponsored by Soprano Brain Alliance, we aren't going down tomorrow.

    Let's get going!

    Xi is hosted on Github, where you can browse all the individual components and see what is available. Feel free to fork, but we appreciate if you can contribute your improvements!
